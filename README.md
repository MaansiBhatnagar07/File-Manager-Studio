# 📁 File Manager Studio

A modern CRUD File Manager built with Python and Streamlit.

## ✨ Features
- 📄 Create Files
- 📖 Read Files
- ✏️ Update Files
- 🗑️ Delete Files
- ⚡ Interactive Streamlit UI

## 🛠️ Tech Stack
- Python
- Streamlit
- Pathlib

## 🚀 How to Run

```bash
pip install streamlit
streamlit run app.py
